# 项目名称
Minecraft 服务器管理面板

## 项目简介
一个基于Flask的Minecraft服务器管理面板，旨在简化服务器管理任务，特别是处理白名单申请。

## 功能
- 用户注册和登录
- 白名单申请审核
- 玩家数据查看和修改
- 排行榜查看
- more

## 技术栈
- Python 3.8+
- Flask 框架
- SQL Server 数据库
- HTML/CSS/JavaScript 前端技术
